¿QUÉ DIFERENCIA A JAVASCRIPT DE CUALQUIER OTRO LENGUAJE?

La gran diferencia entre JavaScript y otros idiomas es que JavaScript es el único lenguaje de programación que nuestro navegador puede leer directamente. Por esta razón, es que podemos escribir y ejecutar nuestro código JavaScript en nuestro propio navegador, sin necesidad de tener ninguna herramienta, sistema o aplicación instalados en nuestro ordenador de forma adicional.
Esta característica hace que este lenguaje sea versátil y que muchos de los programas y aplicaciones más potentes y conocidos, sean escritos en JavaScript.
También hay que destacar que JavaScript está en casi todo el mundo de desarrollo, ya que de un forma u otra siempre se utiliza código de este lenguaje, porque proporciona muchas alternativas y usos que podemos integrar en nuestros programas.

PASO 1: Abrimos nuestro navegador.
PASO 2: Hacemos click derecho y escogemos la opción Inspeccionar y después hacemos click en la parte inferior donde dice: Consola.
PASO 3: Ahora ya podemos escribir nuestro código JavaScript en la consola.
¿CUÁLES SON LOS TIPOS DE DATOS JS?

En JavaScript existen ocho tipos de datos. Mantén el cursor encima de cada tipo de dato y descubre sus ejemplos 😀.

  • 1. Números: let age = 22;
  • 2. BigInts: const bigInt = 1456284536789389443n;
  • 3. Booleanos: let my_truth = True; o let my_not_truth = False;
  • 4. Strings o cadenas: let welcome = "Hello World.";
  • 5. Valor Nulo: let my_null = null;
  • 6. Indefindo (undefined): let notDefinied;
  • 7. Símbolos: let my_sym = Symbol('click event');
  • 8. Objetos: let my_object = {"nombre": "Juan", "edad": 30};
¿CUÁLES SON LAS FUNCIONES DE STRINGS MÁS COMUNES EN JS?

Para ver las diferentes funciones que existen para manipular cadenas en JavaScript, lo primero que vamos a hacer es crear una cadena/string llamado my_string con el que poder ver diferentes ejemplos.

const my_string = 'Hello World';

FUNCIÓN 1: Saber el caracter que está alojado en un index/posición específico. Su sintaxis es: .charAt(n); donde n es el index que queremos buscar. En nuestro ejemplo queremos averiguar cuál es el caracter en la posición 6:

console.log("El carácter en la posición 6 es: ", my_string.charAt = (6));

¡ AVISO IMPORTANTE ! Debemos aclarar, que en JavaScript como en cualquier otro lenguaje, las posiciones empiezan contando desde el 0, incluyendo este último. En nuestro ejemplo:

H = posición 0
e = posición 1
l = posición 2
l = posición 3
o = posición 4
etc...

FUNCIÓN 2: Concatenar otro string o un texto a nuestro string actual. Su sintaxis es: .concat(' mi texto') o .concat(mi_otro_string);. Vamos a poner 2 ejemplos, el primero será concatenar nuestro string con un texto, y el segundo será concater nuestro string con otro string.

FORMA 1: Concatenar nuestro string con un texto:

console.log(my_string.concat(', Welcome!'));

FORMA 2: Concatenar nuestro string con otro string, para ello creamos el nuevo string llamado my_other_string y luego llamamos a la función concatenar.

const my_other_string = ', its a beautiful day!';
console.log(my_string.concat(my_other_string));

FUNCION 3: Buscar un string dentro de nuestro string. Su sintaxis es .includes('string a buscar');. Al igual que en la función anterior, podemos buscar un string nuevo ee incluso un string ya declarado. Esta función devuelve True si encuentra el string, y False si no lo encuentra. Vamos a ver otros 2 ejemplos.

FORMA 1: Buscar un string cualquiera dentro de nuestro string.

console.log(my_string.includes('World'));

FORMA 2: Buscar un string ya declarado dentro de nuestro string.

const find_my_string = 'great';
console.log(my_string.includes(find_my_string));

¡ AVISO IMPORTANTE ! Esta función tiene en cuenta las mayúsculas y minúsculas, por lo que al hacer la búsqueda el string buscado debe ser exactamente igual que donde se busca teniendo en centa mayúsculas y minúsculas. Veamos los ejemplos siguientes.

console.log(my_string.includes('Hello'));
console.log(my_string.includes('hello'));

FUNCION 4: Devuelve True o False si nuestro string empieza con el string dado. Su sintaxis es .startsWith('palabra'); . Como en las funciones anteriores, también se puede buscar por string nuevo y por string ya declarado.

console.log(my_string.startsWith('Hello'));

¡ AVISO IMPORTANTE ! Esta función también tiene en cuenta las mayúsculas y minúscúsculas.

FUNCION 5: Repite el string que tenemos tantas veces como solicitemos. Su sintaxis es .repeat(n); donde n es el número de veces que queremos repetir nuestro string. Veamos un ejemplo.

console.log(my_string.repeat(5));

FUNCION 6: Reemplaza una palabra de nuestro string con la palabra que queramos. Su sintaxis es .replace('palabra_a_reemplazar', 'palabra_nueva');

console.log(my_string.replace('World', 'Sunshine'));

Existen muchas otras funciones que podemos utilizar para manejar los strings en JavaScript, si quieres descubrir más haz click aquí.

¿QUÉ ES UN CONDICIONAL?

Las condicionales en programación son como las decisiones que tomamos en la vida cotidiana. Nos permiten elegir diferentes caminos en función de ciertas situaciones. Por ejemplo, imagina que estás decidiendo si llevar un paraguas o no. Si está lloviendo, decides llevarlo; si no, decides dejarlo en casa. En programación, las condicionales hacen lo mismo: evalúan una situación (como si está lloviendo o no) y ejecutan cierto código según el resultado (llevar o no llevar el paraguas).
Esto nos da control sobre cómo se comporta nuestro programa en diferentes escenarios.
Vamos a ver los 3 tipos de condicionales más utilizados en JavaScript.

Condicional if:

Permite ejecutar un bloque de código si una condición especificada es verdadera. Si la expresión dentro de esta condición es evaluada como verdadera, se ejecutará la instrucción definida dentro del bloque if; en caso contrario, si la expresión es falsa, la instrucción no se ejecutará.

const my_number = 19;
if (my_number >= 18) {
console.log("You are enought old to drive.");
}
const my_number = 13;
if (my_number >= 18) {
console.log("You are enought old to drive.");
}

En nuestro ejemplo, si el número es mayor o igual que 18, nos imprimirá en pantalla: You are enought old to drive, y si es menor que 18, no imprimirá nada.

Condicional if - else:

La declaración else se ejecuta únicamente si la condición evaluada por la declaración if resulta ser falsa. En otras palabras, cuando la instrucción dentro del if no se ejecuta debido a que la condición es falsa, entonces la instrucción dentro del bloque else es la que se ejecuta.
Esto proporciona una bifurcación en la lógica del programa, permitiendo definir acciones alternativas cuando la condición inicial no se cumple.

const my_number = 19;
if (my_number >= 18) {
console.log("You are enought old to drive.");
else {
console.log("You are not enought old to drive.");
}}
const my_number = 12;
if (my_number >= 18) {
console.log("You are enought old to drive.");
else {
console.log("You are not enought old to drive.");
}}

En este ejemplo, si el número dado es mayor o igual que 18, nos imprimirá: You are enough old to drive, y si es menor que 18, nos imprimirá: You are not enough old to drive.

Condicional if - else if - else:

Este tipo de condicinales, funcionan igual que los anteriores, con la peculiaridad, de que teniendo en cuenta que if y else sólo se pueden usar 1 vez y por esta razón solo podemos tener 2 condiciones, else if se puede usar tantas veces como queramos dándonos la oportunidad de incluír más que 2 condiciones.
La sintaxis es igual, pero el orden es:
1. if
2. else if
3. else

Vamos a poner un simple ejemplo:

const mi_edad = 20;
if (mi_edad < 18) {
console.log("Eres menor de edad.");
} else if ( mi_edad > 18 && mi_edad < 65) {
console.log("Eres mayor de edad pero aún no estás jubilado.");
else {
console.log("Eres mayor de edad y ya estás jubilado.");
}}

En este ejemplo, tenemos 3 condiciones, el primero es si mi_edad es menor que 18, entonces nos devolverá "Eres menor de edad", el segundo es que la edad debe comprender entre los números 18 y 65, y en este caso nos imprimirá "Eres mayor de edad pero aún no estás jubilado", y la tercera condición es para el resto de números, es decir los que todavía no hemos recogido con nuestras 2 condiciones anteriores, es decir todos los números que sean mayores que 65, y nos imprimirá "Eres mayor de edad y ya estás jubilado.".

¿QUÉ ES UN OPERADOR TERNARIO?

El operador ternario se podría decir que es un atajo de la condicional if - else. Es decir es una forma más sencilla de declarar y devolver la respuesta a nuestras condiciones.
Como es un if - else, sólo podemos tener 2 condiciones como en los casos originales.

Su sintaxis en JavaScript es: const resultado = (condicion) ? valor_si_verdadero : valor_si_falso; donde resultado es la variable donde guardaremos nuestra respuesta para luego poder imprimirlo.

Vamos a poner un ejemplo claro:

const num = 11;
const mensaje = (num > 0) ? "El número es positivo" : "El número es negativo";
console.log(mensaje)
const num = -12;
const mensaje = (num > 0) ? "El número es positivo" : "El número es negativo";
console.log(mensaje)

En nuestro ejemplo, tenemos un número, y queremos saber si ese número es positivo o negativo, por lo que escribiremos la condicion, si num es mayor que 0, imprímeme "El número es positivo" , en caso contrario, imprímeme "El número es negativo.".

¿DECLARACIÓN DE FUNCIÓN O EXPRESIÓN DE FUNCIÓN?

Declaración de función y expresión de función son 2 formas diferentes de crear funciones.

Vamos a poner 1 ejemplo de cada una para entender mejor sus diferencias.

La principal diferencia entre una declaración de función y una expresión de función es cómo se asigna y se utiliza en el código. En una expresión de función, la función se asigna a una variable, mientras que en una declaración de función, el nombre de la función se utiliza directamente para llamarla.

En resumen, la elección entre una declaración de función y una expresión de función depende de la situación particular y de las necesidades de legibilidad, flexibilidad y reutilización del código en un contexto específico.

¿QUÉ ES LA PALABRA CLAVE 'this'?

En JavaScript, la palabra clave this siempre se refiere a un objeto. Lo que pasa es que el objeto al que se refiere variará dependiendo de cómo y dónde se llame this.
Hay varias formas de usar la palabra clave this, en esta sección vamos a ver cómo se utiliza y cómo se comporta cuando se llama dentro de un método de un objeto.

En ese caso hace referencia al objeto en el que se llamó el método.

¡ AVISO ! hay que aclarar que this no es una variable, es una palabra clave, por lo que su valor no se puede cambiar ni reasignar.

Usar "this" dentro de un método de un objeto permite acceder a las propiedades y métodos de ese objeto de manera dinámica.

¿Y por qué usar this y no utilizar directamente el nombre de nuestras variables?

Porque "this" se utiliza para referirse al objeto actual dentro de un método de un objeto, mientras que {usuario} por sí solo no tiene un significado específico en este contexto y dará como resultado un error de referencia no definida.
